# MIND - Plataforma de Bienestar Emocional

Este proyecto es un MVP funcional de una plataforma de salud mental diseñada para jóvenes y profesionales.

## Características
- **Check-in Diario**: Registro de estado de ánimo, ansiedad, energía y sueño.
- **Alertas Inteligentes**: Detección automática de situaciones de riesgo (ansiedad > 8 o tendencias negativas).
- **Recursos Locales**: Directorio bilingüe (Castellano/Euskera) con integración de llamadas y mapas.
- **Panel Profesional**: Seguimiento detallado de pacientes, estadísticas y gestión de alertas.
- **Apoyo Inmediato**: Ejercicios de respiración guiada y chat de apoyo simulado con IA.
- **Gamificación**: Sistema de puntos por constancia en el bienestar.

## Tecnologías
- Angular 18+
- CSS Moderno (Glassmorphism & Zen UI)
- LocalStorage para persistencia de datos (preparado para Firebase)
- Responsive Design (Mobile-First)

## Instrucciones de ejecución
1. `npm install`
2. `npm start`
3. Abrir `http://localhost:4200`
